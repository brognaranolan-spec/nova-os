Brancher des analytics réelles (agrégées, multi-visiteurs)
1. Crée un compte gratuit sur GoatCounter (goatcounter.com) ou Plausible.
2. Donne-moi le code de ton site → je colle le script de suivi dans le site (2 min).
3. Les vrais chiffres s'affichent alors dans leur tableau de bord. Les achats passent par e-mail (mailto) ; pour les compter automatiquement, on peut brancher un formulaire (Formspree) qui fournit le nombre d'envois.

Mettre en ligne

  1. Tes modifications sont déjà visibles en aperçu sur le site depuis ce navigateur (recharge index.html).
  2. Clique Télécharger : tu obtiens le fichier products.js à jour.
  3. Remplace js/products.js du projet par ce fichier, puis redéploie (ou demande-moi de pousser la mise en ligne).
  4. En ~1 min, les changements sont visibles par tous les visiteurs.
ⓘ Aperçu vs publication. « Aperçu » = visible seulement sur cet appareil (brouillon local). « Publication » = le fichier products.js déployé, vu par tout le monde. Le bandeau en haut indique si un brouillon est en attente.
